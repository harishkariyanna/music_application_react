# Music Application - React & .NET

A full-stack music streaming application built with React (Frontend) and .NET (Backend).

## Features
- Music streaming and playback
- User authentication and authorization
- Media upload functionality
- Playlist management
- Subscription plans
- Admin dashboard
- YouTube integration

## Tech Stack
- **Frontend**: React, TypeScript, Material-UI
- **Backend**: .NET Core, Entity Framework
- **Database**: SQL Server

## Setup Instructions

### Backend
1. Navigate to the backend directory
2. Create `appsettings.json` with your database connection string and credentials
3. Run `dotnet restore`
4. Run `dotnet ef database update`
5. Run `dotnet run`

### Frontend
1. Navigate to `music_player_front` directory
2. Run `npm install`
3. Run `npm start`
